# Projeto Java no IntelliJ IDEA

Este é um guia passo a passo para executar um projeto Java criado no JetBrains IntelliJ IDEA.

## Pré-requisitos

Antes de começar, verifique se você possui o seguinte software instalado em seu sistema:

- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html)
- [JetBrains IntelliJ IDEA](https://www.jetbrains.com/idea/)

## Clonando o Repositório

Clone este repositório para o seu sistema local usando o Git:

```shell
git clone https://github.com/lennrique/SenacPOO.git
```
## Abrindo o Projeto no IntelliJ IDEA

1. Abra o IntelliJ IDEA.
2. Clique em "File" (Arquivo) no menu superior.
3. Selecione "Open" (Abrir) e navegue até a pasta onde você clonou o repositório.
4. Selecione a pasta do projeto e clique em "OK".

O IntelliJ IDEA agora abrirá o projeto.

## Configurando o SDK do Projeto

1. No IntelliJ IDEA, clique com o botão direito no diretório do projeto.
2. Selecione "Open Module Settings" (Abrir Configurações do Módulo).
3. Na seção "Project", verifique se a versão do JDK está configurada corretamente.
4. Caso não esteja, clique em "New" (Novo) e selecione a pasta do JDK instalado em seu sistema.

## Executando o Projeto

1. Abra a classe principal do projeto Main.java.
2. Precione Shift + F10 para executar

O IntelliJ IDEA executará o projeto e você verá a saída no console.
